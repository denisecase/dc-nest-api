## Installation

```pwsh
npm install
```

## Running the app

```pwsh
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```pwsh
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
